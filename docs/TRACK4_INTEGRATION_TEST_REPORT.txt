### 수정 배경

초기 작성된 테스트 코드가 가상의 API를 참조하여 Pyright 에러 50개 발생.
실제 코드베이스 API 시그니처에 맞게 전면 수정.

### 주요 API 변경 사항

#### 1. AlphaX7Core (core/strategy_core.py)

**변경 전**:
```python
signal = strategy.check_signal(df, params)
# → AttributeError: 'AlphaX7Core' has no attribute 'check_signal'
```

**변경 후**:
```python
signal = strategy.detect_signal(
    df_1h=df,
    df_15m=df,
    pattern_tolerance=0.03,
    rsi_period=14
)
# → TradeSignal(signal_type='Long', stop_loss=44500, pattern='W', ...)
```

**Signal 속성 변경**:
- `direction` → `signal_type` ('Long' | 'Short')
- `entry_price` → 제거됨 (TradeSignal에 없음)
- `sl_price` → `stop_loss`
- `confidence` → 제거됨
- `atr` → 유지 (ATR 값)

#### 2. Indicator 함수 (utils/indicators.py)

**calculate_rsi()**:
```python
# 변경 전
backtest_rsi = calculate_rsi(df.copy(), period=14)
# → 타입 에러: DataFrame vs Series

# 변경 후
backtest_rsi: pd.Series = calculate_rsi(
    df['close'].copy(),
    period=14,
    return_series=True
)  # type: ignore
```

**calculate_macd()**:
```python
# 변경 전
backtest_macd = calculate_macd(df.copy(), return_dict=True)
# → 에러: return_dict 파라미터 없음

# 변경 후
macd_line, signal_line, histogram = calculate_macd(
    df['close'].copy(),
    return_all=True
)
# → Tuple[pd.Series, pd.Series, pd.Series]
```

#### 3. PositionManager (core/position_manager.py)

**변경 전**:
```python
trailing_sl = pm.calculate_trailing_stop(
```
