2ï¸âƒ£ API Key Registration

  1. Click the Settings tab at the top
  2. Enter API Key in the exchange card
  3. Enter Secret Key
  4. (OKX/Bitget only) Enter Passphrase
  5. Click Test Connection button to verify
  6. Click Save to save settings
If you see "Connected!" with your balance displayed, you're all set.

3ï¸âƒ£ Data Download

  1. Click the Data tab
  2. Select exchange (e.g., Bybit)
  3. Select symbol (e.g., BTCUSDT)
  4. Click Download button
  5. Wait for download to complete (1-2 minutes)
You must download data before running backtest or optimization!

4ï¸âƒ£ Backtest

  1. Click the Backtest tab
  2. Select exchange and symbol
  3. Click Load to load data
  4. Select a Preset (optimized settings recommended)
  5. Click Run Backtest

5ï¸âƒ£ Optimization

  1. Click the Optimization tab
  2. Set parameter ranges (default recommended)
  3. Choose search method:
    • Fast Search: Quick, approximate results
    • Full Search: Slow, precise results
  4. Click Start button
  5. Review optimal results when complete
  6. Click Save as Preset to save
Optimization results are saved in the user/presets/ folder.

6ï¸âƒ£ Live Trading

  1. Click the Dashboard tab
  2. Select exchange
  3. Select symbol (e.g., BTCUSDT)
  4. Set investment amount (USDT)
  5. Set leverage (3-10x recommended)
  6. Select Preset
  7. Click Start Bot
