🎯 Alpha-X7 전략 개요

Alpha-X7은 W/M 패턴을 기반으로 한 추세 추종 전략입니다.

핵심 특징

  • 📈 W 패턴 (이중 바닥): Long 진입 신호
  • 📉 M 패턴 (이중 천장): Short 진입 신호
  • 🔍 RSI 필터: 과매수/과매도 확인
  • 🌐 MTF (Multi-Timeframe): 상위 타임프레임 추세 정렬
  • 🎚️ ATR 기반 SL/TP: 변동성 반영 손익 설정
  • 🔄 트레일링 스탑: 수익 보호

🟢 진입 조건

Long 진입 (매수)

조건 기준
패턴 W 패턴 (이중 바닥) 감지
RSI < 35 (과매도 상태)
MTF 추세 상위 TF에서 상승 추세
유효시간 패턴 감지 후 설정 시간 내
모든 조건이 충족되면 다음 봉 시가에 진입합니다.

Short 진입 (매도)

조건 기준
패턴 M 패턴 (이중 천장) 감지
RSI > 65 (과매수 상태)
MTF 추세 상위 TF에서 하락 추세
유효시간 패턴 감지 후 설정 시간 내

📐 W/M 패턴 설명

W 패턴 (이중 바닥)


         /\
        /  \    /
       /    \  /
      /      \/
     L1      L2

L1: 첫 번째 저점

L2: 두 번째 저점 (L1과 비슷하거나 약간 높음)

→ L2 이후 상승 반전 예상 → Long 진입

M 패턴 (이중 천장)


      H1      H2
     /  \    /  \
    /    \  /    \
   /      \/      \

H1: 첫 번째 고점

H2: 두 번째 고점 (H1과 비슷하거나 약간 낮음)

→ H2 이후 하락 반전 예상 → Short 진입

파라미터 설명 기본값 범위
filter_tf 추세 필터 타임프레임 4h 2h, 4h, D
atr_mult SL 거리 (ATR 배수) 1.25 1.0 ~ 2.0
trail_start_r 트레일링 시작점 (R 배수) 1.0 0.5 ~ 2.0
trail_dist_r 트레일링 거리 (R 배수) 0.2 0.1 ~ 0.5

🔄 트레일링 스탑

작동 원리

  1. 진입 후 가격이 유리한 방향으로 이동
  2. trail_start_r × Risk 만큼 이익 발생 시 트레일링 시작
  3. 이후 가격이 계속 유리하면 SL이 따라 이동
  4. 가격이 trail_dist_r × Risk 만큼 되돌아오면 청산

예시

진입가: $100
SL: $95 (Risk = $5)
trail_start_r = 1.0
trail_dist_r = 0.2

가격이 $105로 상승 (1R 이익)
→ 트레일링 시작
→ 새 SL = $104 (최고점 - 0.2R)

가격이 $108로 상승
→ 새 SL = $107

가격이 $107로 하락
→ SL 도달, 청산 (이익 $7)
