📖 사용 가이드

1️⃣ 설치

  1. TwinStar_Quantum_Setup_v1.0.0.exe 실행
  2. 설치 경로 선택 (기본: C:\TwinStar)
  3. 바탕화면 아이콘 생성 (선택)
  4. 설치 완료 후 프로그램 실행
삭제 시 API 키, 프리셋, 수집된 데이터는 보존됩니다.

2️⃣ API 키 등록

  1. 프로그램 상단의 Settings 탭 클릭
  2. 사용할 거래소 카드에서 API Key 입력
  3. Secret Key 입력
  4. (OKX/Bitget만) Passphrase 입력
  5. Test Connection 버튼 클릭 → 연결 확인
  6. Save 버튼으로 저장
"Connected!" 메시지와 함께 잔고가 표시되면 성공입니다.

3️⃣ 데이터 다운로드

  1. Data 탭 클릭
  2. 거래소 선택 (예: Bybit)
  3. 심볼 선택 (예: BTCUSDT)
  4. Download 버튼 클릭
  5. 다운로드 완료까지 대기 (1-2분 소요)
백테스트와 최적화 전에 반드시 데이터를 다운로드해야 합니다!

4️⃣ 백테스트

  1. Backtest 탭 클릭
  2. 거래소, 심볼 선택
  3. Load 버튼으로 데이터 로드
  4. Preset 선택 (최적화된 설정 사용 권장)
  5. Run Backtest 클릭

결과 지표 해석

지표 설명 권장 기준
Trades 총 거래 횟수 30회 이상
Win Rate 승률 55% 이상
Simple Return 단순 수익률 양수
Compound Return 복리 수익률 양수
MDD 최대 낙폭 20% 이하

5️⃣ 최적화

  1. Optimization 탭 클릭
  2. 파라미터 범위 설정 (기본값 권장)
  3. 검색 방식 선택:
    • Fast Search: 빠름, 대략적인 결과
    • Full Search: 느림, 정밀한 결과
  4. Start 버튼 클릭
  5. 완료 후 최적 결과 확인
  6. Save as Preset으로 저장
최적화 결과는 user/presets/ 폴더에 저장됩니다.

6️⃣ 실매매

  1. Dashboard 탭 클릭
  2. 거래소 선택
  3. 심볼 선택 (예: BTCUSDT)
  4. 투자 금액 설정 (USDT)
  5. 레버리지 설정 (3-10x 권장)
  6. Preset 선택
  7. Start Bot 클릭
⚠️ 주의: 처음에는 반드시 소액으로 테스트하세요!
봇이 실행 중일 때는 프로그램을 닫지 마세요. 포지션이 열려있으면 수동으로 관리해야 합니다.
