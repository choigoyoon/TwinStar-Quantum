#### 2. 데이터 자동 매칭
**현재**: 수동으로 데이터 파일 선택  
**개선**: 심볼 입력 시 자동 데이터 매칭

```python
def _auto_match_data(self, symbol):
    pattern = f"*{symbol.lower()}*.parquet"
    matches = list(cache_dir.glob(pattern))
    if matches:
        self.data_combo.setCurrentText(matches[0].name)
```

## 📌 탭별 상세 개선안

### 1. 매매탭 (TradingDashboard)

| 현재 | 개선 |
|------|------|
| 프리셋 수동 선택 | 자동 추천 (승률 기준) |
